#### docent.DBService.advisory_lock

```python
advisory_lock(fg_id, action_id)
```

Acquires a PostgreSQL advisory lock for the given FrameGrid ID and action ID.

This provides a concurrency safety mechanism that can prevent race conditions
when multiple processes or tasks attempt to modify the same FrameGrid data.

**Parameters:**

- **fg_id** (<code>[str](#str)</code>) – The FrameGrid ID to lock
- **action_id** (<code>[str](#str)</code>) – An identifier for the action being performed

<details class="example" open>
<summary>Example</summary>
```python
async with db_service.advisory_lock(fg_id, "compute_filter"):
    # This code is protected by the lock
    await db_service.compute_filter(fg_id, filter_id)
```
</details>

#### docent.DBService.upsert_dim

```python
upsert_dim(fg_id, dim, upsert_filters=True)
```

Generalized dimension upsert method that handles creating or updating dimensions and their filters.

**Parameters:**

- **fg_id** (<code>[str](#str)</code>) – Frame grid ID
- **dim** (<code>[FrameDimension](#docent._frames.filters.FrameDimension)</code>) – The dimension to upsert
- **upsert_filters** (<code>[bool](#bool)</code>) – Whether to update filters as well. If True, will delete and recreate filters that have changed.
